Quickstart

The easiest way to get to know the player is by using the setup wizard. Select an example, set the file or playlist you want to play and copy-paste the embed code to your site. If you want to play Youtube content in the player, you also have to copy the yt.swf file along with the player-viral.swf.

Wiki and source code

The JW Player Wiki contains a wealth of information about the player, including:

  1. The roadmap with full changelogs for each version.
  2. All supported file formats (and playlists).
  3. All supported flashvars (for customizing the player).
  4. All supported API calls (for actionscript / javascript interaction).
  5. All supported skinning elements (for creating your own graphics).

The source code of all different versions of the player can be found here. You can click a version and download the ZIP files (the small links at the bottom!).

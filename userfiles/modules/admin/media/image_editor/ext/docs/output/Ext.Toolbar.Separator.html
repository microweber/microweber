Class Ext.Toolbar.Separator

Package:Ext
Defined In:Toolbar.js
Class:Toolbar.Separator
Extends:Toolbar.Item
A simple class that adds a vertical separator bar between toolbar items. Example usage: 
new Ext.Panel({
	tbar : [
		'Item 1',
		{xtype: 'tbseparator'}, // or '-'
		'Item 2'
	]
});

Class Ext.Toolbar.TextItem

Package:Ext
Defined In:Toolbar.js
Class:Toolbar.TextItem
Extends:Toolbar.Item
A simple class that renders text directly into a toolbar. 
new Ext.Panel({
	tbar : [
		{xtype: 'tbtext', text: 'Item 1'} // or simply 'Item 1'
	]
});

Class Ext.Toolbar.Spacer

Package:Ext
Defined In:Toolbar.js
Class:Toolbar.Spacer
Subclasses:Toolbar.Fill
Extends:Toolbar.Item
A simple element that adds extra horizontal space between items in a toolbar. 
new Ext.Panel({
	tbar : [
		'Item 1',
		{xtype: 'tbspacer'}, // or ' '
		'Item 2'
	]
});
